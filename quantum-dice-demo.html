What you are watching

This connects to Quantum Dice Ltd — a photonic chip that uses fundamental quantum randomness to produce numbers that are truly unpredictable. If their server requires a certificate, the visualizer falls back to the browser crypto API (sim mode). Same math, different photons.

Purple — QRNG Value (0–255). Teal — Raw Bits. Red — Entropy. The line never settles. The die has no memory.
