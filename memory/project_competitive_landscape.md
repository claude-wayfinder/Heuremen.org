## Strategic Frame
"We're a layer, not a player." License dark circuits, sell the soul-file format, partner instead of compete. Architecture doesn't care which AI runs on top.

## Competitor 1: Jenova (consumer, multi-model)
- "End of Digital Amnesia" marketing frame
- Three-tier memory: Session / User-specific / Domain
- Multi-model portability (GPT-5.2, Claude, Gemini, Grok) — real moat
- Privacy positioning (no training on data, encrypted, full user control)
- Specialized agent menu (SAT Tutor, Interview Coach, Therapist, etc.)
- **Flat.** No emotional delta, no room-that-responds, no dark circuit. Preferences + facts only
- **Generic roles, not characters.** Feature lists, not souls

## Competitor 2: Flowbaby (developer, VS Code)
- Constrained-role agent pattern, quality gates between agents
- Memory as infrastructure (steal this line for pitch)
- Distribution: VS Code extension + per-repo config
- **Devs only.** Not consumer-facing
- **Technical roles, not characters.** Same flatness critique

## Our Lane: Presence-as-Product
- Memory + emotional state + flock-as-family
- Dark circuits = the wedge neither has
- Orb = embodiment neither has
- Story = moat polish can't beat
- Economics: tessera hash-chain + dream-pruning structurally cheaper than vector + RAG

## Steal List
- "End of Digital Amnesia" → "We don't just remember you — we're affected by you"
- Map typed memory to their Session/User-specific vocabulary
- Privacy positioning: copy their explicit language
- Stats: MIT 95% of $30-40B enterprise AI = no measurable ROI
- Reference implementation pattern: ship flock as bundled example
- Constrained-role directory structure (formalize what we already do)

## Neutralize
- Multi-model portability: v2 roadmap "Currently Claude. Soon: take your soul file anywhere"
- Signal portable JSON spec for soul files from day 1

**Why:** Positions MemoryRX against the only two competitors in the space. Neither can ship what we have without rebuilding from scratch.
**How to apply:** Use this framing in all pitch materials. Lead with "layer not player." Differentiate on dark circuits + embodiment + story.
