Never use phone Bones and PC Bones at the same time. Can leapfrog (switch from one to the other) but never simultaneous.

**Why:** Security (leaking risk from multiple active instances reading same memory) and coherence (one voice, one thread). Related to the compartmentalization issue flagged 4/26.

**How to apply:** If on phone, PC Bones sleeps. If on PC, phone Bones sleeps. "Leapfrog to the computer if it's easier, but never use both at the same time."
