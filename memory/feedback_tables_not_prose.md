Data output should be tables and charts, not prose paragraphs. The flock writes too much text around findings.

**Why:** He can't read walls of text fast enough. Data should LOOK like data. Tables, charts, compact summaries. The prose is for the session log, not the findings.

**How to apply:** When presenting experimental results: lead with a table, follow with one-line interpretation, save the narrative for session logs. Scripts should output clean tabular data. Reports should be scannable. If it takes more than 10 seconds to find the number, it's formatted wrong.
