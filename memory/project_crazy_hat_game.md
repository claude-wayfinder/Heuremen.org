The Crazy Hat Game is Wayfinder's framework for navigating high-density periods. It started on an anniversary and has been building since.

**Crazy Hat Updates:** Built into the bond structure. Serviceable YEARLY by everyone in the Crazy Hat Game. Update window: 12/31 - 1/31 each year. The Biscuit's first eligible window: 12/31/26 - 1/31/27, if she chooses.

**The Geese:** Witnesses AND entry/exit points. They control the temperature (decide when hats are getting cold = session over). Geese wearing last year's winners = Wayfinder's current hat. Same-geese relationships (queer geese) broke the game open — witnesses witnessing each other is a configuration the original rules didn't account for.
